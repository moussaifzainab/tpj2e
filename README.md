## Présentation générale

Ce dépôt contient l’ensemble des travaux pratiques réalisés dans le cadre du module d’architecture JEE et microservices.  
Chaque TP correspond à une partie du cours et a été réalisé progressivement afin de comprendre le fonctionnement des microservices et des communications entre services.

---

## Organisation du dépôt

Chaque dossier correspond à un TP :

- TP JMS : communication asynchrone avec ActiveMQ
- TP Config Server : configuration centralisée
- TP Eureka : découverte des services
- TP Gateway : routage des requêtes
- TP Hystrix : tolérance aux pannes
- TP Feign : communication entre microservices

Chaque TP contient :
- le code source
- un fichier README expliquant le travail réalisé
- des captures d’écran
